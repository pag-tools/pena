• Pre-requisites

    • Docker

  • Installation

    1. Download PENA project and extract it on your local environment
    2. Go to PENA folder and build PENA container:
      3. $ cd pena/ && docker build -t pena

  • Running experiments

    • Run PENA container:
      • $ docker run -it -v $(pwd)/pena:/pena -p 8081:8081 --rm pena bash
    • To run our experiments, go to $ cd PENA/experiments and run the bash script desired:
      • $ ./efficiency-single-conflicts.sh
      $ ./efficiency-multiple-conflicts.sh
      $ ./accuracy-precision.sh
      $ ./accuracy-recall.sh
      $ ./wordpress-conflicts.sh
